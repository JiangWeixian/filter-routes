## features

- 📊 Stats - output bag bundlesize
- 📦 Exclude `dependencies` and `peerDependencies`, and bundle `devDependencies`
- 💪 Typescript - type safe
- 🃏 Auto setup package condition exports
- ✅ Vitest

## install

```console
pnpm i filter-routes
```

## development

- **Setup** - `pnpm i`
- **Build** - `pnpm build`
